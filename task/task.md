Необходимо реализовать программу,  которая анализирует лог

Требования разработки
Сборка под Linux через C Make Допустимо использовать только STL
и системные вызовы ОС
C Make должен обеспечивать: DE BUG -
сборка -- с санитайзерами включаем
в сборку через-DC M AKE_BUILD_T YPE
=Debug; RE LE A SE -сборка -- без
санитайзеров
Имена входного файла (-ов) передаются
в программу через аргументы
командной строки
Программа выводит статус процесса
обработки входных данных (прогресс)
Решение должно быть эффективным
по использованию памяти
(подтверждается санитайзерами)
Выходной файл представлен
в C SV-формате
Критерии оценки
Программа собирается
(не допускаются ошибки времени компиляции)
Программа соответствует требованиям по памяти
(8Gb R AM)
Дедлайн предоставления решения -- 16:00 \n
(оценивается по времени загрузки файлов)
Решение считается предоставленным, если
загружено в сетевую папку команды
Предоставление решения раньше дедлайна, будет
иметь значение только при выполнении всех прочих
требований
При защите решения дополнительно могут быть
оценены:
Способность команды обоснавать и (или)
объяснить фрагмент кода, на который указал
эксперт
Вывод санитайзеров
Скорость работы программы
